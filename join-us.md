# เข้าร่วมกับเรา

## ข้อดีของการเข้าร่วมองค์กรนี้

- สอนใช้ Github และทดลองได้ในพื้นที่นี้
- สร้าง Public repo ภายใต้องค์กร `pcshsbr`
- อวดในพอร์ตฟอลิโอว่าเป็นงานโรงเรียน (ได้ไหม)
- คิดไม่ออกแล้ว เท่มั้ง
